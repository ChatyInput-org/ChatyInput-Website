# ChatyInput Website

Product website for [chatyinput.com](https://chatyinput.com).

## Stack

Pure static HTML/CSS/JS. No frameworks, no build step, zero dependencies.

## Local Development

```bash
python3 -m http.server 8901
# Open http://localhost:8901
```

## Deployment

Hosted on GitHub Pages. Pushes to `main` deploy automatically.

- **Domain:** chatyinput.com
- **Analytics:** Cloudflare Web Analytics

## Files

```
index.html          # Main landing page
404.html            # Custom 404 page
logo.png            # ChatyInput logo
favicon-16.png      # Favicon 16x16
favicon-32.png      # Favicon 32x32
apple-touch-icon.png # iOS home screen icon
sitemap.xml         # Sitemap for SEO
robots.txt          # Crawler instructions
CNAME               # Custom domain config
```
